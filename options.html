Clés API (OBLIGATOIRE)

Configurez vos clés API pour les différentes plateformes

Pour obtenir vos clés Twitch :

  1. Rendez-vous sur dev.twitch.tv/console/apps
  2. Cliquez sur "Register Your Application"
  3. Nom : "Nowtify" | OAuth Redirect : http://localhost
  4. Catégorie : "Application Integration"
  5. Copiez le Client ID (identifiant public)
  6. Cliquez sur "New Secret" pour générer le Client Secret (clé confidentielle)

Ces deux clés permettent de générer automatiquement un token OAuth.
